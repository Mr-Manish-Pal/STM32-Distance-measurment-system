# 📌 Overview

This repository demonstrates the complete workflow of programming an **STM32F103C8T6 Blue Pill** microcontroller using the **ST-Link V2 Debugger/Programmer** and **STM32CubeIDE**.

The project is specifically designed for beginners entering the field of **Embedded Systems**, **Firmware Development**, and **ARM Cortex-M Microcontrollers**.

Unlike simple LED blinking examples, this repository focuses on teaching the **complete development workflow** used by professional embedded engineers.

---

# 🎯 Objectives

- Learn STM32 Architecture
- Learn Blue Pill Hardware
- Configure STM32CubeIDE
- Program STM32 using ST-Link V2
- Build Professional Embedded Workflow
- Understand Flashing & Debugging
- Prepare Foundation for Advanced STM32 Projects
